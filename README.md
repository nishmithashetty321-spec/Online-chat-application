# Online Chat Application

A real-time chat application built using Spring Boot, WebSocket, STOMP, Thymeleaf, and Bootstrap. The application allows multiple users to join a chat room and exchange messages instantly without refreshing the page.

### Features
- Real-time messaging using WebSocket
- Multiple users can join the chat
- Instant message delivery
- User join notifications
- Responsive UI using Bootstrap
- STOMP messaging protocol support
- Simple and lightweight architecture
